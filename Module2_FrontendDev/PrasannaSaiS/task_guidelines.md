How to Use This Exercise Book
This exercise book covers Module 2 — Frontend Development Basics for the Digital Nurture 5.0 Deep
Skilling Program. The 10 hands-on exercises are organised to progressively build your frontend skills
— starting from HTML and CSS fundamentals, moving through JavaScript, then into React, Angular,
and Vue.js frameworks, and finally covering accessibility, cross-browser compatibility, and state
management.
All exercises share a single running project — a Student Portal web application — so your work builds
on itself naturally from one exercise to the next.

Submission Guidelines
• Organise solutions in a folder named Module2_FrontendDev/<YourName>/
• Each hands-on gets its own subfolder: handson_01/, handson_02/, ...
• For framework exercises (React, Angular, Vue): include the full project folder minus
node_modules
• Push to your GitHub repo and share the URL with your POC
NOTE Hands-On 1 through 4 are standalone HTML/CSS/JS files — no framework needed. HandsOn 5 onwards require Node.js and npm.

Common Scenario: Student Portal Web Application
All exercises build one unified project — a Student Portal. Each hands-on adds a new layer to this
portal, so by the end you will have a working, multi-page, accessible, responsive frontend application.
 The Student Portal is a web interface where students can view their profile, browse enrolled
courses, check grades, and receive notifications. You will build the UI layer progressively — first with
plain HTML/CSS/JS, then migrate features into React, Angular, and Vue.js to understand how
frameworks solve real problems.

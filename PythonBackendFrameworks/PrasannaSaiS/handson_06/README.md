# Hands-On 6/7 FastAPI Course API

This project implements a FastAPI-based course management API with:
- Pydantic validation
- async SQLAlchemy access
- CRUD endpoints for courses, students, and enrollments
- pagination and filtering for courses
- background task email confirmation
- Swagger UI at /docs

Run with:

```powershell
python -m uvicorn main:app --reload
```
